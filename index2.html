JavaScript Cheat Sheet

JS CHEAT SHEET

Variables

var
Declares a variable with function or global scope.
var name = "John";
let
Declares a block-scoped variable (ES6).
let age = 25;
const
Declares a block-scoped, read-only variable (ES6).
const pi = 3.14;

Data Types

String
Represents text.
let text = "Hello, World!";
Number
Represents numerical values.
let count = 42;
Boolean
Represents true or false values.
let isActive = true;
Array
Stores multiple values in a single variable.
let fruits = ["apple", "banana", "cherry"];
Object
Stores properties and methods.
let person = { name: "John", age: 30 };
Undefined
Variable declared but not assigned a value.
let data;
Null
Represents "no value."
let empty = null;

Operators

Arithmetic Operators
Used for mathematical operations.
let sum = 5 + 3; // addition let diff = 5 - 3; // subtraction
Comparison Operators
Compare two values.
let isEqual = 5 == "5"; // true
Logical Operators
Combine boolean values.
let result = (true && false); // false
Assignment Operators
Assign values to variables.
let x = 10; x += 5; // x = 15

Functions

Function Declaration
Defines a function.
function greet() { console.log("Hello!"); }
Function Expression
Stores a function in a variable.
let sayHi = function() { console.log("Hi!"); };
Arrow Functions
Concise syntax for function expressions (ES6).
let add = (a, b) => a + b;

Control Flow

if...else
Executes code based on a condition.
if (x > 10) { console.log("Greater than 10"); } else { console.log("10 or less"); }
switch
Performs different actions based on cases.
let day = 2; switch (day) { case 1: console.log("Monday"); break; case 2: console.log("Tuesday"); break; default: console.log("Other day"); }
for loop
Loops through a block of code a set number of times.
for (let i = 0; i < 5; i++) { console.log(i); }
while loop
Loops as long as a condition is true.
let i = 0; while (i < 5) { console.log(i); i++; }

DOM Manipulation

document.getElementById()
Selects an element by its ID.
let element = document.getElementById("myElement");
document.querySelector()
Selects the first element matching a CSS selector.
let item = document.querySelector(".myClass");
element.innerHTML
Changes the inner HTML of an element.
element.innerHTML = "New content";
element.style.property
Changes the style of an element.
element.style.color = "red";
element.addEventListener()
Attaches an event handler to an element.
button.addEventListener("click", () => { console.log("Button clicked!"); });

Events

click
Triggered when an element is clicked.
button.addEventListener("click", function() { alert("Button clicked!"); });
mouseover
Triggered when the mouse is over an element.
element.addEventListener("mouseover", function() { console.log("Mouse over element"); });
keypress
Triggered when a key is pressed on the keyboard.
document.addEventListener("keypress", function(event) { console.log("Key pressed:", event.key); });

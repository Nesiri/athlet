HTML Cheat Sheet

This document provides a comprehensive guide to all HTML elements, including their purposes and example usage.

The lang attribute is set to en for English, which improves accessibility and helps browsers interpret language-specific content.

Document Structure Elements

HTML Document Declaration and Root

<!DOCTYPE html>
Defines the document as HTML5.
<html>
The root element for the document, containing all HTML code.
<head>
Contains metadata like the document title and link to CSS files.
<body>
The main content of the HTML document, visible on the webpage.

Metadata Elements

<title>
Defines the document's title shown in the browser tab. Example: Page Title
<meta charset="UTF-8">
Sets character encoding, essential for text display.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Sets responsive view for mobile and desktop screens.
<link>
Links external resources, such as CSS files.
<style>
Embeds CSS within HTML.
<script>
Includes JavaScript code within HTML.

Content Sectioning Elements

<header>
Represents introductory content, often containing navigation.
<nav>
Defines navigation links.
<section>
Groups related content.
<article>
Independent content suitable for reuse (e.g., blog posts).
<aside>
Defines content indirectly related to the main content, like sidebars.
<footer>
Contains footer information like copyright.
<main>
Main content of the document.
<div>
Generic container for grouping elements.
<span>
Inline container for styling text or grouping content within a line.

Text Content Elements

<h1> to <h6>
Defines headings, with h1 as the highest level and h6 as the lowest.
<p>
Defines a paragraph.
<a>
Creates a hyperlink. Example: Link
<br>
Inserts a line break.
<strong>
Defines important text, typically displayed in bold.
<em>
Defines emphasized text, usually italicized.
<small>
Indicates less important text, displayed smaller.
<mark>
Highlights text.
<sub> and <sup>
Displays text as subscript or superscript, respectively.
<blockquote>
Defines a block quote for long quotations.
<pre>
Displays preformatted text, preserving whitespace.
<code>
Represents inline code snippets.

List Elements

<ul>
Defines an unordered (bulleted) list.
<ol>
Defines an ordered (numbered) list.
<li>
Represents an individual list item.
<dl>, <dt>, <dd>
Creates a description list with terms and descriptions.

Table Elements

<table>
Creates a table.
<tr>
Defines a row within a table.
<th>
Creates a header cell.
<td>
Creates a standard data cell.
<caption>
Provides a table caption.
<thead>, <tbody>, <tfoot>
Defines header, body, and footer sections within a table.

Form Elements

<form>
Creates a form for user input.
<input>
Defines an input field. Example:
<label>
Labels an input element.
<textarea>
Creates a multi-line text input.
<select> and <option>
Creates a dropdown list. Example:
<button>
Creates a clickable button.
